Privacy Policy

隐私政策

ShitClaude 的默认设计是本地优先：能在浏览器完成的检测，不上传到服务器。

我们处理哪些数据

浏览器端检测会读取时区、语言、Intl 区域、字体渲染、WebGL 渲染器、数字格式、日期格式、键盘布局、屏幕 DPI、Emoji 渲染宽度等信息。这些计算在你的浏览器里完成。

服务端 IP 检测会读取请求 IP，并向 ip-api.com 查询国家、城市、时区、ISP、代理或数据中心标记。查询结果会返回给当前页面用于计算风险分。

我们不做什么

  • 不要求注册或登录。
  • 不把检测结果写入数据库。
  • 不收集姓名、邮箱、支付信息或 Claude / Codex 账号信息。
  • 不把浏览器指纹卖给第三方。

托管日志

网站部署平台、CDN 或 Serverless 运行环境可能保留常规访问日志，例如 IP、User-Agent、请求路径和时间戳。这些日志用于安全、调试和基础运维，保留周期取决于你的托管服务商配置。

广告与 Cookie

如果网站启用 Google AdSense 或其他广告服务，第三方广告服务商可能使用 Cookie、广告标识符或类似技术展示广告、衡量广告效果并防止无效流量。

Google 可能会使用 Cookie 根据用户此前访问过的网站投放广告。你可以在 Google 的广告设置页面管理个性化广告，也可以在浏览器中限制第三方 Cookie。

开源项目

本项目代码会公开在 GitHub。开源仓库不应包含个人密钥、部署项目 ID、本地 agent 配置、生成视频文件或环境变量。Fork 用户应替换自己的广告账号、隐私政策和部署配置。

联系我们

如果你发现隐私问题或安全风险，请通过 GitHub Issue 联系维护者。
