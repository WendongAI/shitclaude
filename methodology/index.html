Claude / Codex 地区画像检测方法

ShitClaude 不是官方风控结论，而是把浏览器和网络层可见的地区信号摊开给你看。

为什么地区画像重要

现代 AI 服务可以从多个低强度信号里推断用户位置：时区、语言、IP、字体、渲染器、数字格式和日期格式单独看都不决定结论，但组合起来会形成很强的画像。

14 项信号

分数如何理解

0-44：低风险，当前浏览器暴露的地区特征较弱。

45-74：中风险，存在明显地区画像，但还不够一致。

75-100：高风险，多个客户端和服务端信号指向同一地区。

局限性

真实服务端风控还可能包含账号历史、手机号、支付方式、请求模式、设备信誉、滥用记录和网络信誉。ShitClaude 不知道这些内部信号，因此分数只能作为外部可见画像的估计。

开源复核

检测逻辑在 zh/script.js，测试在 tests/script.test.js。你可以在 GitHub 上审阅权重、国家画像和每项命中条件。
